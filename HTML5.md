-   `<audio>`和`<video>`很像，标签之间的元素是不支持显示的，同样支持多个`<source>`
-   拖放：任何元素都可以拖放
    -   未完待续
-   `<canvas>`：
    -   使用 JavaScript 可以在`<canvas>`中绘画
-   `<svg>`可伸缩的矢量图，可以插入矢量图

-   Web 存储：`localStorage`本地存储，存储没有时间限制,`sessionStorage`：暂时存储，关闭浏览器窗口后数据就会被删除
-   添加了许多新的`<input>`类型
